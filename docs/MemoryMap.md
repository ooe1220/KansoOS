| 物理アドレス              | 大きさ   | 用途・説明                                                    |
| ------------------- | ----- | -------------------------------------------------------- |
| **0x0000〜0x03FF**   | 1KB   | **IVT (割り込み表)** |
| **0x0600〜0x7BFF**   | 512B  | **MBR退避領域** <br>BIOSが0x7C00に読み込んだ後にこの場所へ退避               |
| **〜0x7BFF**   | 約30KB | **スタック領域** <br> 保護モード移行処理に使用              |
| **0x7C00〜0x7DFF**   | 512B  | **VBR (ブートローダ)** |
| **0x8000〜**  | 可変    | **カーネル領域** <br>32bitカーネルや初期OSコードを配置                      |
| **0x10000〜**        | 可変    | **ユーザプログラム領域** <br>ユーザアプリを展開                     |
| **0xF0000〜0xFFFFF** | 64KB  | **BIOS ROM** <br>BIOSは自作しているのでORGやF埋めを見ると理解し易い                     |
